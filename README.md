# YouTube Transcript Viewer

YouTubeの動画ページで字幕・文字起こしをサイドパネルにワンクリックで表示するChrome拡張機能です。

---

## 概要

YouTube を視聴しながら文字起こしをサイドパネルで確認できます。外部APIへのアクセスは一切行わず、YouTubeネイティブの文字起こし機能を利用してデータを取得するため、安全かつ高速に動作します。

---

## 機能

| 機能 | 説明 |
|------|------|
| サイドパネル表示 | 画面右側にスライドインするパネルで字幕を一覧表示 |
| タイムスタンプ付き表示 | 各セグメントに再生時間を表示 |
| クリックでシーク | 字幕をクリックするとその時点から再生 |
| 再生位置ハイライト | 現在再生中の字幕行をリアルタイムで強調表示 |
| キーワード検索 | パネル内の検索ボックスで字幕テキストを即時フィルタリング |
| 全文コピー | 字幕テキスト全体をタイムスタンプ付きでクリップボードにコピー |
| SPA対応 | YouTubeのページ遷移（動画切り替え）を自動検知してリセット |

---

## 使用技術

- **Chrome Extensions Manifest V3**
- Vanilla JavaScript（フレームワーク不使用）
- CSS（固定レイアウト・ダークテーマ）
- YouTube ネイティブ文字起こし DOM 操作

---

## インストール方法

Chrome Web Store への公開前のため、デベロッパーモードでの手動インストールが必要です。

1. このリポジトリをクローンまたはZIPでダウンロードして展開します

   ```bash
   git clone https://github.com/<your-username>/youtube-transcript-extension.git
   ```

2. Chrome のアドレスバーに `chrome://extensions` と入力して開きます

3. 右上の「**デベロッパー モード**」をオンにします

4. 「**パッケージ化されていない拡張機能を読み込む**」をクリックし、展開したフォルダを選択します

5. 拡張機能一覧に「YouTube Transcript Viewer」が表示されればインストール完了です

---

## 使い方

1. YouTubeで動画を開きます
2. 画面右下に表示される赤い「**字幕**」ボタンをクリックします
3. サイドパネルが開き、字幕・文字起こしが自動的に読み込まれます

### 各操作

| 操作 | 方法 |
|------|------|
| パネルを開く / 閉じる | 「字幕」ボタンをクリック |
| 特定の場面に移動 | 字幕の行をクリック |
| キーワードで絞り込む | パネル上部の検索ボックスに入力 |
| 全文コピー | パネル右上の 📋 ボタン |
| 再読み込み | パネル右上の ↺ ボタン |

> **注意:** 字幕が存在しない動画では文字起こしを表示できません。

## ファイル構成

```
youtube-transcript-extension/
├── manifest.json   # 拡張機能の設定ファイル（Manifest V3）
├── content.js      # YouTube ページに注入されるメインスクリプト
├── content.css     # サイドパネル・トグルボタンのスタイル
└── background.js   # Service Worker（バックグラウンド処理）
```
